What is this?

This resource converts Hanako's single-player mod NPC Style Cover Animations from gta5-mods.com into a ready-to-use FiveM server resource. All animation clips (.ycd), configuration files (.meta / .ymt) are bundled into a standalone package — download, drop into resources, and it works.

Why I made this

My server needed NPC-style cover animations, but existing FiveM versions were either incomplete or had too many dependencies. So I took the SP version from gta5-mods.com, adapted it for FiveM multiplayer, and built a complete weapon animation override set (60+ weapons supported). Hope this helps others with the same need.

Features

Full NPC-style cover animations: enter/exit, movement, idle — all replaced

Independent 1h (pistol/SMG) and 2h (rifle/sniper) animation sets with custom upper-body

Weapon holding animation replacement (optimized 2h weapon grip)

Combat attributes auto-applied (ability, movement mode, range, etc.)

Covers 60+ weapons — all vanilla GTA5 weapons supported

Standalone resource, zero external dependencies, plug & play

Installation

Put Arthur_Npcstyleanimations folder into your server's resources directory

Add ensure Arthur_Npcstyleanimations to your server.cfg

Restart the server

Uninstallation

Simply delete the folder from resources and remove the ensure line from server.cfg. This resource does NOT modify any server-side saves or databases.

本插件将 Hanako 制作的 SP 模组 NPC Style Cover Animations 中的 NPC 风格掩体动画、武器持握动画、战斗姿态等完整迁移到了 FiveM 平台。所有动画文件（.ycd）、配置（.meta / .ymt）均已打包为独立 FiveM 资源，无需任何外部依赖，下载解压丢进 resources 即可使用。

制作原因

我的服务器需要 NPC 风格的掩体动作，但网上现有的 FiveM 版本要么依赖过多、要么功能不全。因此我将 gta5-mods.com 上的 SP 版本重新整理、适配为 FiveM 资源，并补充了完整的武器动画覆盖集（支持 60+ 种武器），希望也能帮助到有同样需求的人。

功能特点

玩家掩体动作全覆盖：进出掩体、掩体移动、掩体 idle 全部替换为 NPC 风格

1h（手枪/SMG）和 2h（步枪/狙击）武器独立动作集，含自定义上半身动画

武器持握动画替换（2h 武器持握姿势优化）

战斗属性自动应用（战斗能力、移动模式、攻击范围等）

覆盖 60+ 种武器，完整覆盖所有原版 GTA5 武器

独立资源，无外部依赖，即插即用

安装方法

将 Arthur_Npcstyleanimations 文件夹放入服务器的 resources 目录

在 server.cfg 中添加：ensure Arthur_Npcstyleanimations

重启服务器

卸载方法

从 resources 目录删除文件夹，移除 server.cfg 中的 ensure 行即可。本插件不会修改服务器端任何存档或数据库。
